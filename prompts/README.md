# Prompts Log

This folder contains all GPT prompts used during the project.

## Format

Each prompt should be saved with:
- Date
- Purpose (e.g., "Code generation", "Model design", "Paper writing")
- The actual prompt
- Any follow-up prompts

## Example

```
Date: 2025-11-20
Purpose: Code generation - CLIP image feature extraction
Prompt: "Write a Python function that loads an image and extracts features using CLIP model..."
```
